# Applied Statistics for a Tropical Arboretum

This repository contains the datasets, R scripts, and rendered analytical outputs associated with the study:

> **Dendrometric Data from a Tropical Arboretum for Teaching Applied Statistics and Contributing to Living Collection Monitoring**

The study describes the use of dendrometric data from the arboretum of the Rio de Janeiro Botanical Garden Research Institute (JBRJ), Brazil, as a common empirical framework for teaching applied statistics in a graduate course in Botany. The analyses also generate quantitative information that may contribute to the description, monitoring, and management of the living collection.

## Online analyses

Rendered versions of the main analyses can be accessed through the project website:

**[Open the project website](https://jmalonso55.github.io/Applied-Statistics-for-a-Tropical-Arboretum/)**

The HTML documents display the R code, statistical outputs, tables, and figures generated by the corresponding R Markdown files.

## Study and course context

The dendrometric data were collected during a census conducted between 2021 and 2022 in the planted area of the JBRJ arboretum. Plants with diameter at breast height (DBH) equal to or greater than 10 cm were included.

After data checking and preprocessing, the working dataset comprised **4,631 individuals**, including:

* 3,011 trees;
* 1,362 individuals of Arecaceae;
* 258 plants belonging to other growth-habit categories.

The dataset was used in the graduate course **Applied Statistics for Botany**, offered by the National School of Tropical Botany at JBRJ. The course was conducted as an intensive 40-hour module from 25 to 29 May 2026 and included 10 graduate students with different levels of previous experience in R and applied statistics.

The same dataset was used throughout the course to connect data preparation, exploratory analysis, probability, statistical inference, regression models, and model diagnostics with botanical and living-collection questions.

## Data

The `data` folder contains three versions of the dataset:

| File                                                    | Description                                                                                                       |
| ------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| [`raw_data.csv`](data/raw_data.csv)                     | Original census data before the analytical preprocessing workflow                                                 |
| [`pre_processed_data.csv`](data/pre_processed_data.csv) | Data after structural cleaning, recoding, consistency checking, and calculation of derived dendrometric variables |
| [`processed_data.csv`](data/processed_data.csv)         | Final dataset after preprocessing and taxonomic nomenclature updating, used in the statistical analyses           |

The datasets include taxonomic, spatial, and dendrometric information, such as botanical family, genus, species, arboretum planting bed, total height, circumference and diameter at breast height, wood density, number of stems, basal area, and slenderness.

Some individuals were identified only to family or genus level. Taxonomic and growth-habit information should therefore be interpreted according to the level of identification available in the institutional records and the subsequent data-checking procedures documented in the scripts.

## R scripts

The scripts are numbered according to the pedagogical sequence in which the statistical topics were presented during the course.

| Script                                                                          | Main contents                                                                                                                            |
| ------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| [`1_a_data_preprocessing.Rmd`](code/1_a_data_preprocessing.Rmd)                 | Data import, filtering, recoding, consistency checks, processing of multi-stem plants, and calculation of derived dendrometric variables |
| [`1_b_updating_nomenclature.Rmd`](code/1_b_updating_nomenclature.Rmd)           | Taxonomic checking and updating of botanical nomenclature                                                                                |
| [`2_descriptive_analysis.Rmd`](code/2_descriptive_analysis.Rmd)                 | Frequencies, measures of central tendency and dispersion, quantiles, group summaries, and exploratory data visualization                 |
| [`3_probability_random_variables.Rmd`](code/3_probability_random_variables.Rmd) | Empirical probabilities and applications of binomial, Poisson, and normal distributions                                                  |
| [`4_statistical_inference.Rmd`](code/4_statistical_inference.Rmd)               | Confidence intervals, parametric and non-parametric hypothesis tests, group comparisons, and bootstrap procedures                        |
| [`5_regression_models.Rmd`](code/5_regression_models.Rmd)                       | Linear and quadratic regression, generalized linear models, mixed-effects models, model comparison, and diagnostic procedures            |
| [`6_manuscript_code.Rmd`](code/6_manuscript_code.Rmd)                           | Complete analytical workflow used to produce and refine the results, tables, and figures presented in the manuscript                     |

The first five analytical blocks document the progression of the course. The sixth script integrates the analyses selected for the manuscript and includes additional procedures used during the evaluation and refinement of the results.

## Main analytical approaches

The repository includes examples of:

* data cleaning and transformation;
* taxonomic and growth-habit checking;
* processing of plants with multiple stems;
* calculation of equivalent DBH, basal area, and slenderness;
* descriptive statistics and frequency distributions;
* histograms, boxplots, scatterplots, bar charts, treemaps, and bump charts;
* empirical and theoretical probability distributions;
* confidence intervals and hypothesis testing;
* bootstrap procedures;
* chi-square tests and measures of association;
* generalized linear models with binomial distribution;
* linear and quadratic height-DBH models;
* mixed-effects models accounting for variation among arboretum planting beds;
* model comparison using goodness-of-fit measures and information criteria;
* graphical and simulation-based residual diagnostics.

The manuscript-level analyses include comparisons between taxonomic abundance and structural dominance, evaluation of multiple-stem occurrence, and height-DBH models for *Roystonea* and *Carapa*.

## Reproducing the analyses

The repository can be downloaded using the **Code > Download ZIP** option on GitHub or cloned with Git:

```bash
git clone https://github.com/jmalonso55/Applied-Statistics-for-a-Tropical-Arboretum.git
```

Open the desired `.Rmd` file in RStudio and render it using the **Knit** button or the `rmarkdown::render()` function.

For example:

```r
rmarkdown::render("code/6_manuscript_code.Rmd")
```

The manuscript script uses the final processed dataset directly from the repository. An internet connection is therefore required when the data are imported through their GitHub raw-content URL.

The analyses were performed in R and use base R functions and packages including:

```r
tidyverse
skimr
stringi
rgbif
treemapify
cowplot
DescTools
scales
DHARMa
ggeffects
lme4
performance
lmtest
lmerTest
emmeans
```

Additional dependencies used by individual teaching scripts are identified in their respective package-loading sections. Package versions and changes in external taxonomic databases may affect the exact reproduction of nomenclature-checking procedures.

## Interpretation and limitations

The repository provides reproducible statistical workflows based on an authentic living-collection dataset. Nevertheless, the analytical outputs should not be interpreted as direct management prescriptions.

Patterns observed in the data may also reflect:

* planting history;
* taxonomic composition of the collection;
* past management decisions;
* differences among arboretum planting beds;
* environmental and soil heterogeneity;
* incomplete taxonomic identification;
* measurement and inventory limitations.

Statistical results should therefore be interpreted together with field observations, institutional records, site conditions, and collection-management objectives.
